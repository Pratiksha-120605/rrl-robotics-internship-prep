# Mobile Robot Description

Differential-drive mobile robot modeled using URDF/Xacro in ROS 2 Humble.

## Robot Visualization (RViz)

![Mobile Robot RViz](images/mobile_robot_rviz.png)

## Features
- Differential drive base
- Two driven wheels + caster wheel
- Modular Xacro structure
- RViz visualization with launch file
